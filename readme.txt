Automatically adds an image to a Jetpack Tiled Gallery by adding a tag to the image during upload. 

== Description ==

Makes it easy to add an image to a Jetpack Tiled Gallery by adding a media tag to the image during upload. 
This plugin allows you to use the [Media Tags Plugin](https://wordpress.org/plugins/media-tags/) together with the [Jetpack tiled Gallery plugin](http://jetpack.me/support/tiled-galleries/) so you need to have both of those plugins installed for this to work.

== Installation ==

1. Ensure both the Media Tags and Jetpack Tiled Galleries plugins are installed (links in description)
2. Go to your admin area and select Plugins -> Add new from the menu.
3. Search for "Tagged Media Galleries Bridge".
4. Click install.
5. Activate the plugin through the 'Plugins' menu in WordPress
6. Create a new post on your blog and add thiss shortcode [Mediatags_gallery columns="2" type="rectangular" media_tags="Alphabets"]
7. Add an image to your media library and use the Media Tags plugin to tag the image with "Alphabets".
